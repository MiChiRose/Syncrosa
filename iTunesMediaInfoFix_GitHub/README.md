# iTunesMediaInfoFix 
**The ultimate restoration tool for legacy iTunes libraries.**

This project offers two distinct versions tailored for different eras of Mac OS X. Whether you are running a relatively modern High Sierra machine or a truly vintage Snow Leopard setup, we have you covered.

### 🌟 Features
- **Duplicate Album Merger:** Automatically identifies and merges albums split due to slightly different metadata.
- **Apple Metadata Fetcher:** Connects to the Apple Search API to download official Years and Genres for tracks with missing information.
- **Legacy Compatibility:** Works on systems as old as 10.6 (Legacy edition).

### 🛠 How to Use
1. Download the app.
2. Launch and click "Start Restoration".
3. Watch the magic happen in the real-time console.

*Keep the legacy Mac spirit alive!*
