# iGeniusAI Legacy Pro (Objective-C) 🚀 [WIP]

A native, high-performance rewrite of the iGeniusAI legacy track, built from the ground up using **Objective-C** and **Cocoa**.

> [!IMPORTANT]  
> This version is currently in **active development** (Develop Branch). It is not yet recommended for production use, but serves as the future of iGeniusAI for vintage Intel Macs.

### 🎯 Objective
The goal of this project is to replace the Python-based legacy version with a pure native application. This eliminates the need for a Python interpreter and provides:
- **Zero Dependencies:** No more Python environment issues on OS X.
- **Improved Performance:** Faster UI responsiveness and better memory management.
- **Native Look & Feel:** A true Cocoa experience that respects the classic Aqua aesthetic.

### ✨ Features (Planned & In-Progress)
- **AI Playlist Generator:** Native integration with Gemini, Groq, and OpenRouter APIs.
- **Advanced Media Fixer:** High-speed metadata restoration and library organization.
- **Resilient Networking:** Built-in SSL/TLS bypass for uninterrupted API connections on OS X 10.9+.
- **Lightweight Footprint:** Optimized for older hardware (Core 2 Duo / Early Core i series).

### 🛠 Tech Stack
- **Language:** Objective-C
- **Framework:** AppKit (Cocoa)
- **Minimum OS:** OS X 10.9 Mavericks
- **Target Hardware:** Intel-based Macs (2008-2013)

### 📥 How to Build
1. Open `iGeniusAI-legacy-objc.xcodeproj` in Xcode.
2. Select your target Mac as the destination.
3. Build and Run (`Cmd + R`).
